# Agrinho_2026O seu site AgroFuturo 2026 é um projeto web educativo desenvolvido com o objetivo de mostrar como a tecnologia pode ser aplicada na agricultura moderna e na sustentabilidade.

Ele foi construído utilizando principalmente HTML, CSS e JavaScript, que são as linguagens fundamentais para criação de páginas na internet.

O HTML foi usado para organizar toda a estrutura do site. Ele define as partes principais da página, como o menu de navegação, as seções de conteúdo (Sobre, Sustentabilidade, Tecnologia e Quiz), os títulos, textos, botões e o rodapé. É ele que “monta” a página e determina onde cada elemento aparece.

O CSS foi responsável por toda a parte visual do site. Ele define as cores, estilos, tamanhos, espaçamentos e o layout geral. No seu projeto, o CSS cria um visual moderno com tons de verde, que representam a natureza e a agricultura. Ele também adiciona efeitos visuais como sombras, animações e um design responsivo, que adapta o site para celulares. Além disso, o CSS permite o modo escuro, que muda toda a aparência da página para uma versão mais escura e confortável para visualização.

O JavaScript foi utilizado para adicionar interatividade ao site. Ele permite que o usuário interaja com os elementos da página. No seu projeto, ele controla o botão de modo escuro, o contador animado de árvores preservadas e o quiz interativo, que mostra mensagens diferentes dependendo da resposta do usuário. Isso torna o site mais dinâmico e interessante, em vez de apenas uma página estática.

Além das linguagens tradicionais, também foi utilizada Inteligência Artificial (IA) como apoio no desenvolvimento do projeto. A IA ajudou na criação, organização e melhoria do código, além de auxiliar na estruturação do conteúdo e na explicação das ideias do site. Ela serviu como uma ferramenta de apoio para acelerar o desenvolvimento, corrigir erros e melhorar a qualidade geral do projeto, contribuindo para um resultado mais completo e profissional.

De forma geral, o site tem como tema a agricultura sustentável e o uso de tecnologia no campo, mostrando ferramentas modernas como drones, sensores e inteligência artificial aplicadas ao agronegócio. Ele também incentiva a consciência ambiental por meio de informações educativas e um quiz interativo.

Em resumo, o projeto combina HTML, CSS e JavaScript para estrutura, design e interatividade, e contou também com o uso de Inteligência Artificial como apoio no desenvolvimento, resultando em um site
